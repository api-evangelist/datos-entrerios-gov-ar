# Entre Rios Open Data (datos-entrerios-gov-ar)

Entre Rios Open Data is a state government open-data portal for Argentina running **CKAN** (~121 datasets).

**URL:** [https://raw.githubusercontent.com/api-evangelist/datos-entrerios-gov-ar/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/datos-entrerios-gov-ar/refs/heads/main/apis.yml)
